스타일 해부 (참조: 90년대 처절 사랑 발라드 두 축)
① 록발라드 축(5곡): 숨죽인 벌스(클린기타 아르페지오·피아노) → 후렴에서 절규 벨팅으로 폭발, 일렉기타 솔로+스트링, 90년대 드라마틱 서사
② 소울발라드 축(2곡): 피아노·어쿠스틱 중심, 허스키하고 절절한 보컬, 미디엄 슬로우, 절제하다 파이널에서 차오름
③ 정서: "목숨 건 사랑" — 사랑이 목숨보다 앞서는 화자, 직설 선언·간절한 부탁의 어법, 곡마다 다른 관계·장면
④ 여성 보컬 변환: 록=powerful female belting / 소울=husky soulful female, 남성 지칭 태그 배제
⑤ 원곡 가사 웹 대조 완료 — 제목·핵심 소절 겹침 없음 (검증 내역은 맨 아래 검토 기록)

검토 기록 (셀프 검토 + 히트곡제조기 + 원곡 대조)

원곡 가사 웹 대조 (표절 방지, 실제 검색 대조)

「금지된 사랑」: 제목이 가사에 직접 등장하지 않음. 고유 표현(초대장·가족·"무릎 꿇은 세상"·"같은 날에 떠나" 등 결혼식 소재) → 1번곡에서 결혼식·가족 소재 전면 배제, 새벽 첫차 도피 장면으로 설계.
「내사랑 내곁에」: 후렴 "내 곁에 있어줘"(상대에게 남아달라 부탁) 구조 → 3번곡은 "내가 여기 있을게"(내가 남는다)로 방향 자체를 반전. "너마저 떠나면"·"철없는 욕심" 등 고유 소절 미사용. 7곡 제목 모두 원곡들과 동명 아님.

3단계 셀프 검토에서 고친 것

① 우리가 죄라면 — 원곡 소재(결혼식·가족) 겹침 원천 차단, 도피 새벽 장면으로 교체
② 심장도 가져가 — 파이널의 "내가 여기 있어"가 3번곡 제목과 충돌 → "미련이라 부르지 마 사랑이라 불러"로 교체
③ 내가 여기 있을게 — 후렴 마지막 줄이 길어 호흡 끊김 → "눈 뜨면 나를 봐"로 축약
④ 나를 미워해 — 파이널 "너 하나만"이 1번곡 후렴 "너야 너 하나야"와 겹침 → "그 말만은 진짜였어"로 교체
⑤ 다음 생엔 내가 먼저 — 초벌 유지 (반지 낀 손 장면이 킥)
⑥ 나 없는 날들에게 — 파이널 "다음에 만나면"이 5번곡(다음 생) 컨셉과 충돌 → "먼저 가서 자리 맡아둘게요"로 교체
⑦ 돌아온다 했잖아 — 2절 "골목"이 1번곡과 겹침 → 교체. "두 사람 몫 밥짓기" 장치가 기존 발행곡 재가되어도("두 잔의 커피")와 유사 → 휴대폰·현관 장면으로 교체

4단계 히트곡제조기 검토 (곡별 소견)

① 킥 = "너와 사는 평생이 나의 형벌이길" — 벌을 상으로 뒤집는 역설 펀치라인, 파이널에서만 등장
② 제목이 후렴 첫 줄에서 바로 터짐. 심장 → "그게 무거워지거든" → "사랑이라 불러" 3단 연결로 오브제 회수
③ 소울 축 숨 고르기 트랙. "무서운 꿈은 내가 다 꿀게"가 눈물 포인트, 존댓말 아닌 반말로 6번곡과 차별
④ "손톱이 손바닥을 파고들었어" — 설명 대신 장면. 차가운 벌스와 절규 후렴의 낙차가 이 곡의 킥
⑤ "반지 낀 네 손을 보며 웃었어 / 울면 지는 거니까" — 앨범 최강 벌스 줄. 후렴 "생을 건너서라도"가 절규 자리
⑥ 유일한 존댓말 곡. 밥솥·겨울옷·사진 고르기 등 생활 디테일로 처절함 구현, "천천히 아주 천천히 와요" 마무리
⑦ "끝을 내도 네가 와서 끝내야지" 절규 킥. 브릿지의 자기 인정("아는 날이 더 많아")이 어른의 슬픔
전체: 록5(항거→절규→희생→운명→기다림) 사이 3·6번에 소울2(지킴·작별)를 배치해 감정 온도 곡선 확보. 어미는 곡별로 반말 선언·존댓말 당부·독백을 분산.
