  • 2. 무릎이 쑤신다더니 — ①「당신 우산」(사물) ②채택: 원망조 질문 ③「비 오는 날이 반갑다」(역설). 왜: 원망처럼 들리는데 사랑인 말 = 이 채널 킥 공식(슬픔+신남)에 정확히 맞는다. 훅노트 재고 「같이 좀 늙지 그랬어」를 파이널에 심어 회수
  • 3. 그쪽 어깨가 다 젖었지 — ①「기울어진 우산」(사물) ②「지금도 그렇게 씌워주니」(질문) ③채택: 관찰이 회상으로 넘어감. 왜: 앨범에서 관찰 화자는 이 곡 하나. "나는 하나도 안 젖었는데 / 왜 이제 와 내가 젖니" — 시간차로 젖는 구조가 이 곡의 한 방
  • 4. 오늘은 우산 안 쓸래 — ①「우산은 접어두고」(사물) ②「비야 니가 좀 울어줘」(비 의인화) ③채택: 제목이 곧 행동. 왜: ②는 비 의인화가 상투적이고 ①은 밋밋. ③은 제목이 행동이고 이유는 후렴 2번째 줄에서 터진다
  • 5. 마르지가 않아 — ①「사흘째 빨래」(사물) ②「언제쯤 마르니」(질문) ③채택: 빨래도 나도 안 마른다. 왜: 비라는 글자를 한 번도 안 쓰고 젖은 것만 남기는 곡 = 격자의 '비를 안 씀' 값을 이 곡이 담당
  • 6. 비 오는 날은 공쳐 — ①「빗물받이 양동이」(사물) ②「오늘도 공치겠네」(질문) ③채택: 공치는 날인데 안 억울했다는 역설. 왜: 시장 사람들의 실제 말투라 3초 안에 붙는다. '접다'의 중의(가게를 접다 / 우산을 접다)가 파이널을 잡아준다
  • 7. 비 그친 김에 — ①「젖은 운동화 말리고」(사물) ②채택: 친구들 호명 ③「장마 끝났다고 전해라」(역설). 왜: ③의 "전해라"는 아모르파티류 상투어에 가까워 탈락(채널 금지선). ②는 청유형이라 따라 부르기 최고이고 앨범의 문을 닫는다

  • 🔥 1번과 2번의 킥이 같은 말로 시작했다 — 앨범 내부 최대 사고. 두 곡 다 후렴 첫 줄이 "비만 오면"으로 열렸고, 하필 1·2번 연속 트랙이었다. 카탈로그 경고 그대로 "킥이 비슷하면 다른 곡이라도 같은 노래로 들린다" — 최종 린트에서 킥을 전곡 나열해보고 잡았다. → 2번을 "궂은 날엔 무릎이 쑤신다더니"로 교체(12음절 동일, 5060 어휘). 부수 효과가 더 좋다: 2번 후렴에서 '비'라는 글자가 완전히 사라져 격자의 '비 = 몸에 남은 비'(하늘이 아니라 무릎에만 비가 있다) 축이 선명해졌고, '비'는 아웃트로 "비 온다 여보"에서 단 한 번 도착한다. 5번 줄 "나는 아직도 비만 오면"도 "흐린 날이면"으로 함께 교체
  • 3번 허밍 아웃트로 — 1집 「그 노래가 나오면」과 정면 충돌. 초안은 시티팝 104 + "라라라 라라라라" 페이드아웃으로, 1집 곡과 그루브·장치·라라라까지 3중 겹침이었다. → 라라라를 전부 삭제하고 무가사 허밍이 마지막 줄 밑에 깔리는 형태로 교체(스타일 칸도 humming la-la outro → wordless humming outro under the last line). 장면도 라디오·설거지 → 카페 창가·젊은 애들 관찰로 완전히 갈랐다
  • 6번 브릿지 — 3번과 '젖은 어깨'가 겹쳤다. 초안 "하나는 나 주고 하나는 니가 쓰고 / 근데 왜 니가 더 젖었을까"는 3번의 킥을 그대로 반복. → 우산 이미지를 3번에 몰아주고, 6번 브릿지는 셔터를 대신 내려주는 장면으로 교체. 사물이 갈리니 두 곡이 다른 노래가 됐다
  • 6번 브릿지 2차 — '파전'이 1번 부침개와 충돌. 교체안 "파전이나 먹으러 가자고"가 1번의 핵심 사물을 침범 → 삭제. 부침개는 1번에만 남긴다
  • 2번 — 1집 「당신 없이 맞는 봄」과 같은 사별 소재. 각도를 강제로 벌렸다: 그 곡은 벚꽃·존댓말·곱게 슬픔·100BPM 발라드·"약속대로 웃는다" 착지. 2번은 무릎·반말 타박조·118 신스팝·"같이 좀 늙지 그랬어" 착지. 사물·어법·그루브·착지 4축이 전부 달라 같은 노래로 안 들린다

  • 1번 아웃트로 — "잘 먹었습니다"가 강한결 4집 「밥값」 아웃트로와 동일(타 채널이지만 같은 공방 산출물) → "식기 전에 얼른 와 / 어차피 안 올 거면서"로 교체. 뚝 멈춤 장치도 이쪽이 훨씬 세다
  • 2번 프리코러스 — "그게 뭐라고"가 강한결 4집 「아저씨」 브릿지 리프레인 → "왜 그렇게 대충 했을까"로 교체

⑤ 히트곡제조기 통독 (완성본 소견 · 걸린 곳은 고쳤다)
  • 1. 먹을 사람도 없으면서 — 앨범 최강 후보. 킥="한 장은 니 몫으로 부치고 / 식을 때까지 그냥 둔다". 비 오는 날 부침개는 5060 여성에게 설명이 필요 없는 사물이고, 지글지글=빗소리라는 연결이 이 곡을 지탱한다. 브릿지 "손으로 쭉쭉 찢어 먹었지 / 그게 뭐가 맛있다고"가 눈물 지점 — 타박인데 사랑이다
  • 2. 무릎이 쑤신다더니 — 눈물 담당. "먼저 가서 안 아파서 좋아?"는 5060이 실제로 하는 말투다. 브릿지의 유통기한 지난 파스 한 통이 설명 없이 세월을 보여준다. 아웃트로 "비 온다 여보 / 파스 사다 놓을게"에서 곡이 무너진다 — 아카펠라로 뺀 판단이 맞다
  • 3. 그쪽 어깨가 다 젖었지 — 앨범의 품격. 초안 브릿지가 "그립다"류 착한 말이라 약한 줄로 걸렸다 → "왜 말을 안 했냐 물으니 / 그냥 웃기만 하던 사람"으로 교체해 사람을 세웠다. 파이널 "이제는 내가 씌워주고 싶은데 / 우산이 하나뿐이네"가 뒤집는 줄
  • 4. 오늘은 우산 안 쓸래 — 킥은 후렴이 아니라 브릿지에 있다: "근데 이 나이에 어디서 울어 / 화장실도 눈치가 보이는데". 이 두 줄이 곡 전체의 이유다. 파이널 마지막 "내일은 나 또 웃을 거니까"가 없으면 그냥 우는 곡 → 반드시 유지
  • 5. 마르지가 않아 — 가장 조용한데 가장 세다. 킥="마르면 개켜야 하니까 / 개키면 넣어야 하니까 / 넣으면 잊어야 하니까" — 3단 논리 사다리로 왜 안 마르길 바라는지를 설명 없이 보여준다. 비라는 글자 0회를 끝까지 지켰다(검증 완료). 앨범 온도 최저점
  • 6. 비 오는 날은 공쳐 — 자부심 담당. 초안은 순수 인생 축이라 앨범 컨셉(그 사람이 떠오르는 정서)에서 겉돌았다 → "손님도 없고 너도 없고", "비 오면 니가 왔으니까"를 후렴에 심어 사랑 축으로 끌어왔다. 파이널 "공쳐도 나는 안 접었다"의 접다가 가게와 우산에 동시에 걸린다
  • 7. 비 그친 김에 — 앨범을 닫는 문. 따라 부를 줄="울 만큼 울었으면 됐다 / 이제 좀 웃으러 가자". 텐션 상한 점검: 관광버스·트위스트 어휘 0, 무대는 동네와 빨랫줄뿐 → 뽕짝 파티로 안 튄다. 브릿지 "그 사람도 어디선가 / 빨래를 널고 있겠지"가 앨범 7곡의 비를 한 번에 말린다
  • 앨범 곡선 — 온도: 미련(1) → 눈물(2) → 품격(3) → 폭발(4) → 최저점(5) → 자부심(6) → 해방(7). 같은 온도 3연속 없음. 가장 약했던 곡은 6번(컨셉에서 겉돎)이라 후렴을 다시 써서 사랑 축으로 갈아엎었다. 선공개 권장 = 1번(사물의 힘이 제일 즉각적) 또는 2번(눈물 지점이 가장 깊다)
